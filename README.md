# 生活效率工具箱

一个基于 Web 的多功能个人效率管理平台，集成了智能待办清单和专注番茄钟两大工具。

## 功能特性

### 1. 智能待办清单
- ✅ 任务管理（添加、编辑、删除、完成）
- ✅ 分类标签（工作、学习、生活、其他）
- ✅ 优先级设置（高、中、低）
- ✅ 截止日期和提醒时间
- ✅ 多维度筛选和排序
- ✅ 实时统计数据展示
- ✅ 分类占比可视化
- ✅ LocalStorage 数据持久化

### 2. 专注番茄钟
- ✅ 番茄工作法计时器
- ✅ 自定义专注时长（15/25/30/45/60分钟）
- ✅ 短暂休息（5分钟）和长休息（15分钟）
- ✅ 圆形进度环动画
- ✅ 今日、本周、累计统计数据
- ✅ 专注历史记录
- ✅ 提示音（可开关）
- ✅ 浏览器通知
- ✅ 数据持久化

### 3. 工作台架构
- ✅ 顶部 Tab 导航切换
- ✅ 工具间状态保持
- ✅ 数据隔离存储
- ✅ 统一的视觉风格（靛蓝渐变主题）
- ✅ 流畅的交互动效

## 技术栈

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript ES6+
- **UI 框架**: TailwindCSS
- **图标库**: Font Awesome
- **数据存储**: LocalStorage
- **音频**: Web Audio API

## 快速开始

1. 克隆或下载项目
2. 使用任意 HTTP 服务器打开 `index.html`
   ```bash
   # Python
   python -m http.server 8080
   
   # Node.js
   npx http-server -p 8080
   ```
3. 在浏览器中访问 `http://localhost:8080`

## 番茄钟使用说明

### 标准番茄工作法流程
1. 选择专注时长（默认25分钟）
2. 点击"开始专注"按钮
3. 专注时间结束后自动进入休息
4. 每4个专注周期后触发长休息（15分钟）
5. 休息结束后可继续下一个专注周期

### 功能亮点
- **智能计时**: 圆形进度环实时显示剩余时间
- **灵活设置**: 5种预设时长可选
- **数据追踪**: 完整记录每次专注数据
- **声音提醒**: 可开关的提示音
- **浏览器通知**: 计时结束时自动提醒

## 待办清单使用说明

### 核心功能
- **添加任务**: 点击右上角"添加任务"按钮
- **编辑任务**: 点击任务卡片的编辑图标
- **删除任务**: 点击任务卡片的删除图标（需确认）
- **完成任务**: 点击任务前的复选框
- **开始任务**: 将任务状态设置为"进行中"

### 筛选和排序
- 按分类筛选：工作/学习/生活/其他
- 按状态筛选：待处理/进行中/即将到期/已过期/已完成
- 按多种方式排序：创建时间/优先级/截止日期
